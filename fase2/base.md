Agora começa o jogo real.

## O que aprender (ESSENCIAL)

- App Router
- `app/`
- `page.tsx`
- `layout.tsx`
- `Link`
- `useRouter`
- Server vs Client Components
- `use client`
- `fetch` no Next

## Recursos (oficial)

- https://nextjs.org/docs/app

## Ordem exata

1. Criar projeto

```bash
npx create-next-app@latest
```

2. Entender estrutura `app/`

3. Criar rotas

4. Criar layouts

5. Navegação

6. Componentes client

## Exercício

Crie um mini app com ao menos as seguintes rotas:

- `/login`
- `/dashboard`
- `/profile`

Boa prática: comece com uma rota, adicione layout compartilhado e, em seguida, componentes client onde necessário.
